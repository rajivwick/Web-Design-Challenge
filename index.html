Summary Latitude

This project designed to analyze how weather changes as you move closer to the equator. We started by first pulling data using OpenWeatherMap API to construct a dataset on over 500 cities.

Matplltlib was then used to plot multiple weather characteristics against the latitude of the location. The Variables that were assessed were temperature, cloudiness, wind speed and humidity.
